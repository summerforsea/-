# 六轴机械臂 — CAN 电机控制脚本

基于 [42Motor CAN 通信协议](CAN协议手册.md)，通过 `cansend` 直接控制六个关节电机。

## 脚本说明

| 脚本 | 命令码 | 作用 |
|------|--------|------|
| `enable.sh` | `0x01` (data=1) | 使能六个电机，进入运行模式 |
| `disable.sh` | `0x01` (data=0) | 失能六个电机，停止运行 |
| `home.sh` | `0x15` | 将当前位置设为零位，自动保存到 EEPROM |

## 使用方法

```bash
# 使能电机
./enable.sh

# 停止电机
./disable.sh

# 设置零位（先将机械臂摆到零点姿态再执行）
./home.sh
```

## CAN ID 映射

CAN ID = `(NodeID << 7) | 命令码`

| 关节 | Node ID | enable/disable | home |
|------|---------|---------------|------|
| joint1 | 1 | `081` | `095` |
| joint2 | 2 | `101` | `115` |
| joint3 | 3 | `181` | `195` |
| joint4 | 4 | `201` | `215` |
| joint5 | 5 | `281` | `295` |
| joint6 | 6 | `301` | `315` |

> 如果实际 CAN 接口不是 `can0`，修改脚本顶部的 `CAN_IF` 变量。
