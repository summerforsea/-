# 传送带模块 (Conveyor Belt Module)

本模块是“水果全自动分拣礼盒一体化机器人系统”的**自动上料核心组件**。由地瓜 RDK X5 上位机主控，通过通信控制下位机，驱动传送带电机运转，实现水果的自动化上料。

---

## ⚙️ 硬件与架构组成

传送带模块主要包含以下核心硬件及文件：
*   **传送带拓展版**：模块的核心电路连接与拓展版设计（内含 `.epro2` 格式原理图/PCB及对应的 `.png` 预览图）。
*   **降压模块**（12降5V）：为系统及单片机提供稳定的电源转换（内含 `12降5V.epro2` 及 `.png` 预览图）。
*   **下位机主控**：STM32C8T6 单片机。
*   **驱动程序**：包含 STM32 电机驱动及与上位机通信控制的核心源码（打包于 `chuansongdai.zip` 中）使用对应的集成开发环境Keil uVision打开单片机工程。

---

## 🛠️ 技术栈与平台依赖

*   **上位机主控平台**：地瓜 RDK X5
*   **下位机微控制器**：STM32C8T6
*   **系统环境**：ROS2 (Humble)
*   **开发环境**：Keil uVision
---

## 🚀 工作原理与运行流程

1.  **指令下发**：RDK X5 上位机根据系统分拣状态，通过通信接口（如串口）向 STM32C8T6 单片机发送控制指令。
2.  **电机驱动**：STM32C8T6 接收到指令后，调用底层电机驱动程序，控制传送带电机启动/停止或调节速度。
3.  **自动上料**：传送带运转将水果源源不断地送入视觉检测与机械臂抓取区域，实现全流程自动化。
